Datasets & Tools

Open-source tool
PipeForge3D
A Python tool for the generation and construction of 3D pipes for industrial applications. It can forge 3D mesh models and point clouds of pipes, from both randomly generated graphs and given graphs in json format of pipe connections, using an available kit of connection-type meshes. Used to build the synthetic dataset for SBP-Net.
